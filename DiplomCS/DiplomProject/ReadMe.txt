Структура проекта:
1. Form1.cs - интерфейс
2. Model.cs - модель и камера
3. Program.cs - запуск приложения
4. ReadMe.txt - описание проекта

Подключенные библиотеки:
Tao.OpenGl.dll;
Tao.FreeGlut.dll;
Tao.Platform.Windows.dll;
GlmNet.dll;
freeglut.dll

Для создания были использованы:
C#;
Windows Forms;
сторонние библиотеки

Приложение предназначено для демонстрации возможностей OpenGL. 

На основном окне имеются два окна для вывода изображения, также есть возможность управления объектом и камерой с помощью клавиатуры. 
Результат таких манипуляций можно увидеть, также есть GroupBox, в которых отображается текущее положение камеры и модели.
Внизу представлены GroupBox, в которых отображаются соответствующие матрицы.
